# EZCalGo
カレンダーベースのタスク管理アプリケーション。直感的なUIで簡単にタスクの管理と予定の調整が可能です。

## デプロイ先URL
https://hide-yama.github.io/EZCalGo/

## 機能
- スケジュールの自動計算
- クリップボードへのコピー機能
- Googleカレンダーとの連携
- 行の入れ替え機能（Command + ↑/↓）
- 開始日時の設定

## 使い方
1. 開始日時を設定
2. タスク名と所要時間（分）を入力
   ```
   読書 30
   ストレッチ 15
   メールチェック 20
   ```
3. 「作成」ボタンをクリック
4. 生成されたスケジュールをコピーまたはGoogleカレンダーに追加

## 技術スタック
- React
- TypeScript
- Vite
- Tailwind CSS
- Google Calendar API
